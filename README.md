# CNN的原理图
### 关于卷积后的图片维度计算公式：(W-F+2P)/S+1，其中W表示输入数据的大小，F表示卷积层神经元的感受野磁村，S表示步长，P表示边界填充数量，比如输入是7*7，filter是3*3，step是1.pading数量是0，那么更据公式有(7-3+2*0)/1+1=5,即输出是5*5，若步长是2则输出是3*3.
